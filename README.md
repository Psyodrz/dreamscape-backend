# Dreamscape Maze - Backend

Multiplayer Lobby Server for Dreamscape Maze

## Tech Stack

- **Node.js** - Runtime
- **Express** - Web framework
- **PeerJS** - Signaling server

## Setup

```bash
npm install
npm start
```

## Configuration

Server runs on port **3000** by default.

## API Endpoints

- WebSocket connections for multiplayer lobby
- PeerJS signaling for P2P connections

## Deployment

Can be deployed to:

- Render
- Heroku
- Railway
- Any Node.js hosting
